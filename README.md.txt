AudioBook Generator Project

Description:
This project converts PDF, DOCX, and TXT documents into audiobook format using AI-based text processing and Text-to-Speech.

How to Run the Project:

1. Install Python on your system.
2. Open Command Prompt inside the project folder.
3. Install required libraries using:
   pip install -r requirements.txt
4. Run the application using:
   streamlit run app.py

Features:
- Upload PDF, DOCX, or TXT files
- Extract text from documents
- Convert text into audiobook-style narration
- Generate and download MP3 audio file

Technologies Used:
- Python
- Streamlit
- pdfplumber
- python-docx
- gTTS
