# AI Audiobook Generator

## Overview
An AI-powered application that converts PDF, DOCX, and TXT documents into high-quality audiobooks using Large Language Models (LLMs) and Text-to-Speech (TTS).

## Features
- Upload PDF, DOCX, and TXT files
- Extract text automatically
- Convert text into speech
- Generate downloadable audiobooks

## Technologies Used
- Python
- Streamlit
- OpenAI API
- Text-to-Speech (TTS)

## Future Improvements
- Multiple voice options
- Chapter-wise audio generation
